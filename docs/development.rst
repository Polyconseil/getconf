Development
===========

Clone the repository and install the development dependencies in a virtualenv:

.. code-block:: sh

    pip install -r requirements_dev.txt

To run tests:

.. code-block:: sh

    pytest

To make a release:

.. code-block:: sh

    fullrelease

.. _zest.releaser: https://zestreleaser.readthedocs.io/en/latest/index.html
